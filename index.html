🔐 Password Manager CLI App

This is a command-line application built in Python that helps you securely store and retrieve passwords using AES encryption.

📦 Features

  • Secure password encryption using the cryptography library
  • Local storage using a JSON file
  • Simple CLI interface

🛠️ How to Use

1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   python main.py
